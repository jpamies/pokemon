# Imatges Pokémon

Aquest directori conté les imatges dels Pokémon per a ús offline.

## Estructura

- Les imatges es descarreguen automàticament de l'API
- Format: PNG amb fons transparent
- Mida: 96x96 píxels (sprites oficials)
- Nomenclatura: `pokemon-{id}.png`

## Cache

Les imatges es guarden localment per millorar el rendiment i permetre l'ús offline.
